<CodeGroup>

```bash Terminal
npx hyperframes add transitions-shader
```

</CodeGroup>

## Usage

After installing, add the block to your host composition:

```html
<div data-composition-id="transitions-shader" data-composition-src="compositions/transitions-shader.html" data-start="0" data-duration="55" data-track-index="1" data-width="1920" data-height="1080"></div>
```
